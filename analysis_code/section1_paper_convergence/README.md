# Section 1 Paper-Convergence Code Snapshot

This folder contains a small copy of the Section 1 COSMOS/COSMOS2020 paper-figure code from the analysis repository.

It is included for provenance only. It does not include private data, FITS files, parquet files, catalogs, or bulk analysis outputs.

The copied code expects the analysis repository layout if rerun locally. The manuscript figures in `figures/` are the selected PDF outputs used by the paper draft.

Section 1 uses a 0.5 arcsec COSMOS2020-DP2 matching radius. Section 2 and Section 5 products may need rerun later for full consistency with that matched sample.
